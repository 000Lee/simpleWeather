# 소개
Simple Weather는 사용자가 원하는 도시를 입력하면, 실시간 날씨 정보를 보여주는 미니멀한 날씨 조회 웹 애플리케이션입니다.
<br/>React 기반으로 제작되었으며, OpenWeather API를 활용해 현재 기온, 날씨 상태, 대기 등을 직관적으로 확인할 수 있습니다.
